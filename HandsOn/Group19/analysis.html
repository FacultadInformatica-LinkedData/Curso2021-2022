Dataset Analysis

Se analizarán los datasets "data2018.csv", "data2019.csv", "data2020.csv" y "datos202108.csv"

Campo Tipo Long Descripcion Problemas
Provincia String 2 Identificador numérico de la provincia No es descriptivo
Municipio String 3 Identificador numérico del municipio No es descriptivo
Estación String 3 Identificador numérico de la estación donde se toman medidas de las emisiones No es descriptivo
Magnitud String 2 Identificador numérico del tipo de gas medido así como las unidades de la medición No es descriptivo
Punto_Muestreo String 10 Identificador numérico del punto de muestreo, es la unión de los identificadores de provincia, municipio, estación y magnitud No es descriptivo
Ano String 4 Año en el que se realizó la medición. N/A
Mes String 2 Mes en el que se realizó la medición N/A
Dx String 2 Valor obtenido en la medición el día x Los valores obtenidos se juntan con el numero del dia
Vx String 1 Validación del valor obtenido el día x Los valores obtenidos se juntan con el numero del dia

Naming strategy

  • Domain: http://spainaq.linkeddata.com/
  • Path: http://spainaq.linkeddata.com/aq/
  • Ontologies: http://spainaq.linkeddata.com/aq/ontologies/
  • Classes: http://spainaq.linkeddata.com/aq/ontologies/[ontologyName]#[className]
  • Properties: http://spainaq.linkeddata.com/aq/ontologies/[ontologyName]#[propertyName]
  • Resources: http://spainaq.linkeddata.com/aq/resources/
  • Individual: http://spainaq.linkeddata.com/aq/resources/[className]#[id]
